# kyori-component-json documentation

Hello! This is the branch holding the documentation and the GitHub Action to publish the docs at <https://docs.winlogon.org/kyori-component-json>.

The documentation uses [mdBook](https://github.com/rust-lang/mdBook) for building all the src/\*.md files. You can build it with `mdbook build` and run it as a server with `mdbook serve`.
